Volume_And_Gain_Control_APIs

General Description

This profile enables a device to expose the controls and state of a device that can control the volume of an audio output such as one or more speakers and Control the peer device audio output as a client.

VCS may include zero or more instances of VOCS, AICS. VCP which is the client side of the volume Control Service consists of APIs to control volume and gain of a peer device. It acts as client for VCS, VOCS and AICS
