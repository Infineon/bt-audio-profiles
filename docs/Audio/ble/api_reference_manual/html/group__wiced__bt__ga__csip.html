Lock Procedure.

Tries to aquire lock from the least rank device to highest rank. If any device responds with lock denied unlocks already locked devices.

Parameters
[in]p_set: csip set member entries to perform lock procedure
[in]num_instances: number of client instances sent
[in]lock_val: lock value to be set
[in]cb: callback returned when the complete lock procedure is completed
[in]p_datapointer : this will be return in given cb

Ordered Access Procedure.

Reads lock value from the least rank device to highest rank. If any device responds with locked, returns with the event WICED_BT_GA_CSIP_ORDERED_ACCESS_PROCEDURE_COMPLETE with status set to the procedure status

Parameters
[in]p_set: csip set member entries to be locked
[in]num_instances: number of client instances sent
[in]cb: callback returned when the complete ordered access procedure is completed
[in]p_datapointer : this will be return in given cb
