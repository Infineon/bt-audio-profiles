Function Name: wiced_bt_a2dp_sink_mute_audio.

To mute/unmute the audio while streaming. Called by the application to mute an audio when playing music. The application sets this function to unmute, to restart playing music. This api is supported only (in Embedded mode) when app and profile are running on an Infineon BT chip such as 20721.

Parameters
[in]enable1 to mute, 0 to unmute
[in]ramp_msramp up/down time in milli seconds
Returns
